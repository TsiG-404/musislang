Musislang

This web app helps you learn languages with the use of music. It translates the lyrics to the selected language, includes recommended grammar for some words, a karaoke feature to improve your vocal skills, and separates the vocals and the beat from the selected song using machine learning.

Directions for use:

1. Take a YouTube link of a song which includes the artist and song name.
2. Select the language you want the lyrics translated to.
3. Click "Slang it" (it may take a while if the song is long or not well known).
4. See the results.
